# 🌐 Packet Routing: IP vs MAC

Welcome to the Packet Routing Simulator! This interactive module is designed to help students understand the fundamental logic behind data transmission on the Internet.

## 🎯 Learning Objectives

By the end of this module, you will logically understand:

The functional difference between an IP Address and a MAC Address.

How packet headers change (and what stays the same) as data travels from router to router.

The real-world mechanisms of Encapsulation and Decapsulation.

## 🚀 How to Use This App

Interactive Simulator: Click "Start Journey" to watch the packet travel. Observe the "Live Packet Header" panel on the right closely—watch how the MAC addresses update at every hop while the IP addresses remain constant!

Code Explorer: Toggle between Python and C++ to see how programmers logically structure a packet in memory.

Think & Link: Before looking at the answer, use the drafting text area to write down your own logical deduction. Formulating your own thoughts is the best way to solidify your understanding.

Self-Assessment: Finish the module by taking the Mini-Quiz at the bottom of the page to verify your concepts.


🛠 Enjoy exploring the hidden pathways of the Internet!
